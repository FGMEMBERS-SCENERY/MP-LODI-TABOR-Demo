[English]###################################################################################

All these folders are in the 'data' or 'fgdata' folder

ATTENTION:
The aircraft folder "/Aircraft/MPCarrier" replace the old one, so save your old folder first,
same for the /Navaids/carrier_nav.dat.gz 

All other files will be placed in the existing folders (/Models/Geometry/Bodensee) is new.

Start as captain on the "Lodi" use this properties:

--aircraft=lodi
--callsign=Lodi
do not select Lodi_Tabor_demo as captain

Start as captain on the "Tabor" use this properties:

--aircraft=tabor
--callsign=Tabor
do not select Lodi_Tabor_demo as captain


Captains select in the Menu Multicarrier the other ferry


Pilots start up with:

--aircraft=f-14b  (or whatever)
--carrier=Lodi
--parkpos=cat     (or H1 as helicopter)
--ai-scenario=Lodi_Tabor_demo
or
--aircraft=f-14b  (or whatever)
--carrier=Tabor
--parkpos=cat     (or H1 as helicopter)
--ai-scenario=Lodi_Tabor_demo

Pilots select in the Menu Multicarrier both or one of the two ferries


Have fun 


[German]####################################################################################

Diese Ordner liegen alle im Hauptverzeichnis 'data' bzw. 'fgdata'

ACHTUNG:
Der Flugzeugordner "/Aircraft/MPCarrier" muss überschrieben werden - deshalb bitte vorher das Original sichern :-)
Ebenso die carrier_nav.dat.gz im Ordner /Navaids

Alle übrigen Dateien in den Hauptordnern koennen an entsprechender Stelle eingefuegt werden.

Der Kapitän auf der LODI startet flightgear mit den Properties:

--aircraft=lodi
--callsign=Lodi
(keine Lodi_Tabor_demo auswählen)



Der Kapitän auf der TABOR startet flightgear mit den Properties:

--aircraft=tabor
--callsign=Tabor
(keine Lodi_Tabor_demo auswählen)


Die Kapitäne müssen sich im Menu gegenseitig bestätigen um sich zu sehen



Die Piloten starten:

--aircraft=f-14b  (oder welches Flugzeug auch immer)
--carrier=Lodi
--parkpos=cat     (oder H1 als Helicopter)
--ai-scenario=Lodi_Tabor_demo

oder

--aircraft=f-14b  (oder welches Flugzeug auch immer)
--carrier=Tabor
--parkpos=cat     (oder H1 als Helicopter)
--ai-scenario=Lodi_Tabor_demo

Als Pilot bestätigt man nach dem ersten Start im Menu beide Faehren


Viel Spaß
